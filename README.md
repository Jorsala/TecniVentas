# TecniVentas
Sistema para llevar un control de Stock y Ventas

Requerimientos
-Control de Bodega (Stock)
  --Productos en Existencia
 
-Control de Usuarios
  --Diferentes Niveles de Accesos
-Control de Reportes (Por Usuario y por Cliente)
-Control de Compras
-Control de Ventas
-Impresion de Recibos
